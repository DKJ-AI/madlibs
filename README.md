# MadLibs

A browser-based MadLibs game with Random story templates. Fill in random words without context, then watch the story reveal itself in glorious absurdity.

## What is MadLibs?

MadLibs is a word game where one player asks others for random words (a noun, a verb, an adjective, etc.) without showing them the story. Once all blanks are filled, the story is read aloud — and the mismatch between the serious template and the random words is what makes it funny.

## Features

- Multiple built-in story templates: Job Interview, Breaking News, Love Letter, Recipe , and many more
- Clean, minimal UI — no installs, no dependencies
- All highlighted words are revealed in context once submitted
- Reset and replay any story instantly
- Fully client-side — works offline after loading
